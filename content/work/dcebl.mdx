## Overview
A responsive, SEO-friendly marketing website built on WordPress with custom layouts and scroll-based motion.

## What I built
- Converted PSD designs into fully responsive HTML layouts with mobile-first principles.
- Built the site with WordPress and Elementor Pro — custom page layouts and interactive sections aligned to brand identity and user flow.
- Integrated and customized plugins for product showcase, lead-capture forms, and performance.
- Added smooth scroll-based animations to guide navigation and boost engagement.
- Cross-browser tested with BrowserStack across Chrome, Firefox, Safari, and Edge.
