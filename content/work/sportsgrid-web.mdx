## Overview
A sports-focused web application with live updates and an interactive, real-time experience. I built the frontend from high-fidelity Figma designs into pixel-perfect, fully responsive UI.

## What I built
- Translated high-fidelity Figma designs into responsive React components with pixel-perfect layout accuracy and design consistency.
- Built custom, reusable components with Tailwind CSS and Material UI, following a component-based architecture for scalability and clean code.
- Configured JW Player for video streaming — seamless embedding and playback of live and on-demand content across devices.
- Conducted cross-browser testing with BrowserStack across Chrome, Firefox, Safari, and Edge.
