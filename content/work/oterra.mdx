## What I built
- Built the OTT frontend in React.js with a focus on performance and maintainability across all major browsers.

- Collaborated with designers and backend developers in an Agile environment, translating Figma into pixel-perfect UI and coordinating on API and data flow.
- Kept the codebase clean and documented using functional components, hooks, and separation of concerns.
