The Android & iOS companion app for SportsGrid — live scores, streaming, and stats in a single React Native codebase.

## What I built
- Translated complex Figma UI into pixel-perfect React Native components, accurate and responsive across Android and iOS.
- Built device-independent layouts with ScaledSheet and react-native-size-matters for consistent UI across phones and tablets.
- Enabled live streaming video for real-time content delivery, integrated cleanly into the UI.
- Used react-native-chart-kit for interactive charts and graphs surfacing live stats and analytics.
- Applied the Animated API with interpolation for subtle scroll-based transitions and animated indicators.

<Callout>Leaned on a focused set of React Native libraries — navigation, toasts, image picking, date/time pickers — to ship features faster without bloating the bundle.</Callout>
